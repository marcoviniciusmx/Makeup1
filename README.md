## 🏠 About the project

Este projeto foi desenvolvido como uma landing page profissional para a **Jessica Menezes Makeup**, com foco em fortalecer sua presença digital, transmitir sofisticação e aumentar a conversão de visitantes em contatos.

A proposta do site é apresentar os principais serviços de maquiagem, reforçar autoridade por meio de depoimentos e diferenciais, além de facilitar o agendamento através de botões estratégicos de contato.

O layout foi pensado para oferecer uma navegação agradável, visual refinado e boa experiência tanto no desktop quanto no mobile.

---

## 🧰 Features

- Header fixo com navegação por âncoras
- Hero section com proposta de valor e CTA
- Seção “Sobre” com apresentação da profissional
- Galeria de ambientes e imagens do espaço
- Seção de serviços com cards visuais
- Bloco de benefícios e diferenciais
- Depoimentos de clientes
- Área de contato com acesso rápido
- Botões de WhatsApp e Instagram
- Scroll suave entre seções
- Menu mobile responsivo
- Animações de entrada ao rolar a página
- Estrutura preparada para futuras expansões

---

## 🏗️ Layout and design decisions

O projeto utiliza uma identidade visual elegante e feminina, com tons dourados, bege e marrom, buscando transmitir sofisticação, acolhimento e exclusividade.

Alguns pontos de destaque no layout:

- uso de paleta refinada e coerente com o nicho de beleza
- cards com sombra suave e cantos arredondados
- organização visual clara das seções
- foco em legibilidade e hierarquia visual
- adaptação para diferentes tamanhos de tela
